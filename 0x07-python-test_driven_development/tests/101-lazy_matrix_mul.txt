How To: Usage of 101-lazy_matrix_mul.py
(this is like a readme for the above file)
(i.e means that is / for example)

This library contains one function ``lazy_matrix_mul()``, 
that multiplies 2 matrices

Valid Cases:

	>>> lazy_matrix_mul = __import__('101-lazy_matrix_mul').lazy_matrix_mul
	>>> m_a = [[1, 2]]
	>>> m_b = [[3, 4], [5, 6]]
	>>> print(lazy_matrix_mul(m_a, m_b))
	[[13 16]]

i.e

	>>> m_a = [[3.1, 1.2]]
	>>> m_b = [[0.3, 1.5], [3.0, 0.2]]
	>>> print(lazy_matrix_mul(m_a, m_b))
	[[4.53 4.89]]

i.e

	>>> m_a = [[1, 2]]
	>>> m_b = [[0.25, 0.15], [2, 0.5]]
	>>> print(lazy_matrix_mul(m_a, m_b))
	[[4.25 1.15]]
